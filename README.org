This is a teasing project, mainly for learning golang. It have about 80% of features in [[https://github.com/jarun/nnn][nnn]], [[id:https://github.com/gokcehan/lf][lf]], [[https://github.com/sxyazi/yazi][yazi]]. It does not support multiple tabs and 

* Package used:
tui:
[[https:https://github.com/charmbracelet/bubbletea][bubble tea]]
[[https:https://github.com/charmbracelet/lipgloss][lip gloss]]

chinese named file handling:
[[https:https://github.com/aquilax/truncate][truncate]]

Dependency for preview: chafa(image rendering), ffmpeg(convert vedio to image), libreoffice(convert office file to image), poppler(convert pdf to image)

If you are using kitty, then there will be high resolution preview for images

* Configuration
It will creat a default configuration in "$HOME/.config/tui-fm/config.json", set book mark with full path name, and editor you prefer

* Usage:
1. "j", "k" or "C-n", "C-p" for up and down;
2. "enter" for go down dictionary or xdg-open;
3. "v" for preview;
4. "C-s", "C-v" for searching name;
5. "C-j" for jumping to dictionary;
6. "space" for marking files;
7. "y" for yank file;
8. "p" for past file;
9. "R" for opening editor buffer to rename;
10. "r" for renaming a single file;
11. "c" for copy current file to "place";
12. "n" for creating new file/folder/symlink;
13. "t" for opening a shell in current folder;
14. "b" for jumping to your bookmakr;
15. "M-x", ":" for input command;
16. "M-x" commands:
    - "sh": execute shell command;
    - "up"/"down": to move focus up/down n;
    - "go": to do line n;
    - "goto": go to folder;
    - "copyto": copy current file/folder to "place";
